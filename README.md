# Me Maratha — Account Deletion Page

This page is hosted on GitHub Pages for app store compliance. Users can visit this page to learn how to delete their Me Maratha account and associated data.

## Deploy to GitHub Pages

1. **Create a new GitHub repository** (e.g., `mimaratha-account-deletion`)

2. **Push this folder's contents to the repo:**
   ```bash
   cd account-deletion-page
   git init
   git add .
   git commit -m "Add account deletion page"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/mimaratha-account-deletion.git
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to your repo → **Settings** → **Pages**
   - Source: **Deploy from a branch**
   - Branch: `main` / Folder: `/ (root)`
   - Save

4. **Your page will be live at:**
   ```
   https://YOUR_USERNAME.github.io/mimaratha-account-deletion/
   ```
